# CLI-Net (pipe.net) 🚀

![Project Version](https://img.shields.io/badge/Version-1.0-brightgreen)
![Go Version](https://img.shields.io/badge/Go-1.21+-blue)

**CLI-Net** — это современная терминальная социальная сеть, работающая исключительно через SSH. Здесь нет веб-интерфейса или регистрации по паролю. Аутентификация проходит по вашим SSH-ключам, а общение — через функциональный 4-панельный TUI.

---

## 🌟 Основные возможности

### 🖥️ 1. 4-Панельный### Основные возможности
- **`CHANNELS` (Справа):** Навигация по каналам. Переключение — `Tab`.
- **`ONLINE` (Справа):** Список активных пользователей в канале.
- **`FEED` (Центр):** Окно чата с поддержкой Markdown и изображений в Kitty-режиме.
- **`OSC 52 Copy`:** `Ctrl + Y` копирует последнее сообщение в буфер обмена.

### 🖼️ 2. Инлайн-изображения и GIF
Поддержка **Kitty Graphics Protocol** позволяет видеть картинки прямо в терминале:
- Используйте `/img <url>` для шаринга контента.
- Работает в **Kitty, WezTerm, Ghostty** и др.
- Фоновая загрузка не блокирует интерфейс.

### 👑 3. Иерархия Ролей
- **Владелец (`owner`) 👑:** Полный доступ, создание каналов, управление админами.
- **Администратор (`admin`) ★:** Модерация (кик/бан) и управление темами.
- **Пользователь (`user`):** Базовое общение и смена внешнего вида.

## 🕹️ Управление и Команды

| Горячие клавиши | Действие |
| :--- | :--- |
| **`Tab`** | Следующий канал |
| **`Ctrl + P`** | Панель профиля и настроек |
| **`Ctrl + Y`** | Копировать последнее сообщение (OSC 52) |
| **`Ctrl + C`** / **`Esc`** | Выход |

### 💬 Слэш-команды (`/`)
*Доступные команды зависят от вашей роли (см. `/help`).*

- `/nick <name>` — Сменить никнейм.
- `/color <name/hex>` — Сменить цвет ника (палитра в Профиле).
- `/img <url>` — Отправить картинку/GIF.
- `/clear` — Очистить чат.
- `/topic <text>` (Admin+) — Тема канала.
- `/kick /ban /unban <user>` (Admin+) — Модерация.
- `/newchan <name>` (Owner) — Новый канал.
- `/op /deop <user>` (Owner) — Управление админами.
- `/setowner <user>` (Owner) — Передать права владельца.

## 🚀 Запуск и Подключение

### 1. Сервер (Linux)
```bash
go build -o clinet ./cmd/clinet
sudo systemctl restart clinet
```

### 2. Клиент
```bash
./clinet-connect
```

## 🛠️ Технологии
- **Go** (Wish, Bubble Tea, Lip Gloss, Glamour).
- **Kitty Graphics Protocol** для графики.
- **SQLite** для хранения данных.

## 🛡️ Безопасность
- Авторизация по **SSH Public Keys**.
- Файлы базы данных (`.db`), логи и ключи автоматически скрыты через `.gitignore`.
- Владелец защищен от любых действий модерации.
